# 《Vocabulary Speedrunner》期中進度報告

## 一、 計畫基本資訊
*   **計畫名稱**：基於本機端大語言模型及自然語料庫之動態外語學習工具：以阿拉伯語學習為例
*   **計畫代表人**：鄭裴禎 (國立政治大學 阿拉伯語文學系)
*   **指導老師**：馬穆德 老師
*   **核心理念**：結合大數據詞頻分析與本地 AI 導師，打造隱私安全且高效率的「Speedrun」式單字學習體驗。

## 二、 目前執行進度分析 (完成度：約 75%)

根據計畫書規劃之進度，目前專案已提前完成核心引擎開發，進入初步測試階段。

### 1. 語料庫建置與資料分析 (100% 完成)
*   **已完成事項**：
    *   開發 `parse_wiki.py`：成功實現自動處理阿拉伯語維基百科全量資料 (Wikipedia Dump)。
    *   開發 `frequency_analyzer.py`：建立具備權重梯度的基準單字資料庫 (`voc_database.csv`)，並導入 `qalsadi` 進行初步分詞與停用詞處理。
*   **成果**：成功從真實世界語料中提取詞頻資訊，打破傳統教材與現實脫節的限制。

### 2. 核心程式開發與 AI 整合 (90% 完成)
*   **已完成事項**：
    *   開發 `main_quiz.py`：實現基於詞頻的動態抽選演算法 (Weights-based SRS)。
    *   **AI 批改整合**：成功透過 API 呼叫本地運行的 Ollama (Gemma:2b) 模型，實現語意層級的答案判定。
*   **突破**：系統不再僅能比對字串，而能理解近義詞並容忍合理的拼寫偏差 (如 `سكرا` 與 `شكرا`)。

### 3. 測試與介面優化 (20% 完成)
*   **現況**：目前已具備功能完善的命令列介面 (CLI)，能流暢執行學習循環。
*   **待辦**：GUI 圖形化介面的開發與更多使用者的封閉測試。

## 三、 面臨之挑戰與解決方案

### 1. 阿拉伯語詞形變化複雜 (Morphological Complexity)
*   **挑戰**：阿拉伯語具備複雜的根詞 (Root) 變化，簡單的 Tokenization 難以精確統計同根詞頻。
*   **解決方案**：已引入 `qalsadi` 函式庫輔助，未來將強化 NLP 預處理流程，提升詞頻分析的精確度。

### 2. 本地 LLM 的判定準確度與效能平衡
*   **挑戰**：2B 等級的輕量模型在理解深層語意時偶有幻覺，且在非 GPU 環境下反應速度較慢。
*   **解決方案**：優化 Prompt Engineering (提示工程)，明確規定 AI 僅回傳 "Correct" 或 "Incorrect"；並評估引入更輕量的 1B 模型或優化推理參數。

### 3. 資料庫自動翻譯功能缺位
*   **挑戰**：目前詞頻資料庫僅有阿拉伯文，中文翻譯仍需手動輸入或依賴外部單字表。
*   **解決方案**：規劃整合開源字典 API (如 Glosbe) 進行自動翻譯填充，降低使用者建立單字表的門檻。

## 四、 下階段執行計畫
1.  **GUI 開發**：利用 Python 框架 (如 Tkinter 或 Flet) 製作輕量化視覺介面。
2.  **演算法微調**：根據遺忘曲線 (Ebbinghaus) 優化權重衰減邏輯。
3.  **封閉測試**：邀請阿文系同學參與測試，收集對 AI 批改準確度的回饋。

# 《Vocabulary Speedrunner》海報展設計大綱

## 【標題區】
*   **大標**：Vocabulary Speedrunner：AI 本地化外語學習引擎
*   **副標**：詞頻導航 x AI 家教 x 絕對隱私

## 【亮點模塊 1：為什麼要 Speedrun？】
*   **痛點**：傳統單字書按字母排序，浪費時間在低頻詞；APP 批改死板，寫錯一個符號就算錯。
*   **解法**：用維基百科告訴你什麼單字最重要，用 Gemma AI 像真人一樣改你的作業。

## 【亮點模塊 2：核心技術流程】
*   **(圖示化流程)**：
    1.  **Big Data**: 爬取 Wiki 阿拉伯語全文 (Gensim)。
    2.  **Analysis**: 計算詞頻，賦予每個單字「價值權重」。
    3.  **AI Engine**: Ollama 本機運行，100% 資料不外流。
    4.  **Learning**: 動態抽考，錯愈多、考愈勤。

## 【亮點模塊 3：實機展示】
*   **(截圖預留位)**：展示 CLI 介面下，AI 如何判定「近義詞正確」。
*   **文字說明**：即使使用者回答了同義詞，AI 老師也能理解並給分，大幅降低學習挫折感。
